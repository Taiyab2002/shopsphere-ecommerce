#  ShopSphere – Modern E-Commerce Website

![HTML](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)

![Bootstrap](https://img.shields.io/badge/Bootstrap-5.3-7952B3?style=for-the-badge&logo=bootstrap&logoColor=white)

A modern responsive e-commerce website built using **HTML, CSS, Bootstrap 5, and Vanilla JavaScript**.

This project was developed as part of a **Web Programming course** and demonstrates a complete front-end shopping experience with product browsing, wishlist, shopping cart, checkout, and order processing.

#  Features

###  Home Page

- Modern responsive homepage
- Hero section
- Featured products
- Categories section
- Newsletter section

###  Shop

- Product listing
- Search products
- Category filter
- Price sorting
- Responsive product cards

###  Wishlist

- Add to wishlist
- Remove from wishlist
- Wishlist counter
- Wishlist page

###  Product Details

- Product information
- Quantity selector
- Add to cart
- Buy Now button
- Related products

###  Shopping Cart

- Add products
- Increase quantity
- Decrease quantity
- Remove item
- Order summary
- Live total calculation

###  Checkout

- Billing information
- Multi Currency (USD / BDT)
- Dynamic payment methods
- Delivery estimation
- Order summary

###  Order Success

- Order confirmation
- Customer information
- Payment method
- Order ID
- Purchased products
